# Sample GST invoices

Add 3–5 **anonymized** PDF invoices here for local testing, e.g.:

- `sample-gst-invoice-1.pdf`
- `sample-purchase-bill.pdf`

Use the `/try` page to upload them without signing in (requires `OPENAI_API_KEY`).

The built-in **Run sample invoice** button uses mock JSON data when you only need to demo the UI.
